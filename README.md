# Legend Class Autonomous Cutter

1:96 scale RC model of a USCG Legend Class National Security Cutter with autonomous navigation and bathymetric sonar mapping.

## Platform

- ESP32 dual-core controller
- PCA9685 16-channel PWM driver
- Flysky FS-i6X transmitter with iBUS receiver
- 4S 5000mAh LiPo, dual BEC power system
- Twin 550 motors, differential thrust steering
- BN-220 GPS + SparkFun ICM-20948 9-DOF IMU
- JSN-SR04T depth sonar for bathymetric mapping
- DFPlayer audio with PAM8403 amp
- WiFi AP mode for phone telemetry
