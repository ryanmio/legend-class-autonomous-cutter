01

Capabilities

NAV / 01

GPS waypoint nav

Captures each waypoint within a set radius (~2–3 m), then advances to the next — running a self-driven route with no hand on the sticks.

SONAR / 02

Bottom mapping

A hull-mounted transducer logs depth against position while underway, building a bathymetric picture of the bottom as it goes.

IMU / 03

Compass heading

Magnetometer heading with gyro stabilization; the accelerometer logs roll, pitch, and wave motion through the run.

LINK / 04

Live telemetry

Position, heading, depth, battery, and system state stream over WiFi to a companion iOS app — including gap backfill after a dropout.

POWER / 05

Battery monitoring

INA219 current + voltage sense on the 4S pack. State logged continuously; low-voltage events visible in the telemetry stream.

BILGE / 06

Bilge watch & auto pump

Leak detection with an automatic bilge pump — the boat looks after its own hull while it works.

FW / 07

OTA firmware

Updates go on over the air. The hull stays sealed and watertight — no cracking it open for USB access between runs.

DECK / 08

Deck gun pan

Positional servo on the main gun driven by a dedicated knob channel — slews the turret independently of the helm.

RADAR / 09

Rotating radar

Mast TRS-3D dish driven by a burst-mode PWM loop that simulates slow radar-sweep rotation from a high-speed motor.

Who has control

Three states, one rule — the transmitter sits at the top of the priority stack. Flip the mode switch and the human takes over instantly; lose the RC link and the boat fails safe on its own. Below, that arbitration, on a loop.

How it steers

05

How it maps the bottom

In AUTO the boat runs a grid of waypoints, and a hull-mounted echo sounder takes one reading directly beneath it at each GPS fix — logged against position. The track itself builds the area into a depth map. Below, that survey, simulated.
